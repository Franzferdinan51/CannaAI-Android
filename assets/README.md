# Assets Directory

This directory contains all the static assets used by the CannaAI Flutter application.

## Directory Structure

- `images/` - Image assets for the app UI and content
- `icons/` - App icons and UI icons
- `fonts/` - Custom fonts for the application

## Instructions

1. Add your actual images and icons to the respective directories
2. Update `pubspec.yaml` to include new assets
3. Ensure all images are optimized for mobile (use WebP format when possible)
4. Follow Android and iOS design guidelines for icons and images

## Image Guidelines

- Use PNG format for icons and images with transparency
- Use JPG format for photographs without transparency
- Optimize images for mobile (keep file sizes small)
- Provide multiple resolutions for different screen densities when needed
- Use 2x, 3x suffixes for high-DPI displays
