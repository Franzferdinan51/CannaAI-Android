## Icon Design Guidelines

### Material Design Principles
- Follow Google's Material Design guidelines
- Use consistent stroke weight (2dp for most icons)
- Maintain 24dp grid system for navigation icons
- Use simple, recognizable shapes

### Color Scheme
- Primary: Green (#4CAF50)
- Secondary: Blue (#2196F3)
- Warning: Orange (#FF9800)
- Error: Red (#F44336)
- Success: Green (#4CAF50)
- Neutral: Gray (#757575)

### Style Consistency
- Line icons for navigation (outline style)
- Filled icons for actions when selected
- Consistent stroke width and corner radius
- Good contrast for accessibility

## File Formats
- **PNG** for all icons (supports transparency)
- **SVG** versions for scalability (optional)
- **WebP** for better compression (optional)

## Tools for Creating Icons
- Android Studio's Image Asset Studio
- Material Design Icons (https://fonts.google.com/icons)
- Figma or Adobe Illustrator
- Canva (for quick mockups)
- IcoMoon (for icon fonts)

## Delivery Required
Provide all icons in the following sizes:
- mdpi (1x, 24dp = 24px)
- hdpi (1.5x, 24dp = 36px)
- xhdpi (2x, 24dp = 48px)
- xxhdpi (3x, 24dp = 72px)
- xxxhdpi (4x, 24dp = 96px)

Android Studio can automatically generate these from the source vector assets.
